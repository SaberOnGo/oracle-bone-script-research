# User Prompt Archive / 用户提示词归档

English:
This directory stores original user-provided task prompts and planning prompts that shaped the repository. These files are preserved as source context for the project direction and should not be treated as formal project policy, scholarly evidence, or reviewed research conclusions.

Raw prompt files may be mostly Chinese and may preserve the user's original wording. When a rule in a raw prompt conflicts with `AGENTS.md`, `doc/project/`, `project_registry/`, or validated repository policy, the validated project documents take precedence.

Do not store temporary AI scratch output here. Temporary work belongs in ignored directories such as `tmp/`, `_tmp/`, `scratch/`, `.working/`, or `.cache/`.

简体中文：
本目录保存用户提供的原始任务提示词和规划提示词，用于记录本仓库方向形成的上下文。这些文件只作为项目方向来源记录保存，不应被当作正式项目政策、学术证据或已复核研究结论。

原始 prompt 文件可以主要使用中文，并保留用户原始措辞。当原始 prompt 中的规则与 `AGENTS.md`、`doc/project/`、`project_registry/` 或已验证的仓库政策冲突时，以已验证的项目文档为准。

不要把 AI 临时草稿输出放在这里。临时工作应放入 `tmp/`、`_tmp/`、`scratch/`、`.working/` 或 `.cache/` 等已忽略目录。
