# Corpus Validation Tools / 语料校验工具

English:
This numbered tool area is for repository and corpus validation. The current skeleton validator lives at `tools/validation/check_repository_skeleton.py` so Python tests can import it through a stable package path.

简体中文：
本编号工具区用于仓库和语料校验。当前骨架校验器位于 `tools/validation/check_repository_skeleton.py`，方便 Python 测试通过稳定包路径导入。
