# External Source Prefixes / 外部来源前缀

English:
External source prefixes keep paths short and make provenance recognizable.

Prefixes in `003_external-source-prefixes.csv` cover both external authority abbreviations and project-local staging prefixes. Staging prefixes such as `hust-obc-cat`, `obimd-main`, `obimd-sub`, `evobc-cat`, `ihp-mus-obj`, `cam-hopkins-y`, and `collection-prov` are provenance handles only; they do not promote dataset or collection-page rows into accepted oracle-character, component, inscription, or formal evolution-correspondence records.

简体中文：
外部来源前缀用于缩短路径，并让来源出处一眼可识别。

`003_external-source-prefixes.csv` 同时记录外部权威/专业来源缩写和本项目暂存前缀。`hust-obc-cat`、`obimd-main`、`obimd-sub`、`evobc-cat`、`ihp-mus-obj`、`cam-hopkins-y`、`collection-prov` 等暂存前缀只用于来源追溯；它们不会把数据集行或馆藏页面行提升为已接受的甲骨单字、构件、卜辞或正式字形演化对应记录。
