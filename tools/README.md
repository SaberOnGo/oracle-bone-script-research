# Tools / 工具

English:
This directory contains repository validation, import, graph generation, statistics, and AI context-pack tools. Use `tools/validation/validate_ai_agent_evidence_packs.py` to validate AI Agent evidence-pack drafts after they are generated or edited.

简体中文：
本目录保存仓库校验、导入、图谱生成、统计和 AI 上下文包工具。AI Agent 证据包草稿在生成或修改后，应使用 `tools/validation/validate_ai_agent_evidence_packs.py` 校验。
