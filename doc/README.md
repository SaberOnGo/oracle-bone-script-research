# Documentation / 文档

English:
This directory contains project documentation for humans and AI agents. Human-facing documents should be bilingual in English and Simplified Chinese, or have a clearly linked English/Chinese companion file.

简体中文：
本目录存放给人类和 AI agent 阅读的项目文档。面向人类的文档应中英双语，或提供清楚互链的英文/中文配套文件。

## Areas / 区域

- `project/`: stable project policies and research design.
- `project/`：稳定的项目政策和研究设计。
- `public/user_plan/`: user-facing planning drafts.
- `public/user_plan/`：用户可读的规划草案。
- `public/user_research/`: user and AI Agent research drafts.
- `public/user_research/`：普通用户和 AI Agent 的研究草稿。
