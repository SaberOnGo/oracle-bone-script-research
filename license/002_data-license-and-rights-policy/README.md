# Data License And Rights Policy / 数据许可证与权利政策

English:
Structured metadata can often be shared more safely than raw images or copyrighted text, but each source still needs review. Use `rights_status` fields in metadata and registry files.

简体中文：
结构化 metadata 通常比原始图片或受版权保护文本更容易公开，但每个来源仍需复核。metadata 和 registry 文件中应使用 `rights_status` 字段。
