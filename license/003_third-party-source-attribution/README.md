# Third-Party Source Attribution / 第三方来源标注

English:
Record third-party catalogs, databases, books, papers, museums, and websites here when they are used as sources. If full-text copies are preserved, mark source provenance, rights status, and risk notes.

简体中文：
使用第三方字编、数据库、图书、论文、博物馆或网站作为来源时，在此记录引用信息。不要保存权利不明确的全文副本。
