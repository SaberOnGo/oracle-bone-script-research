# Corpus / 语料主体

English:
This directory is the human-maintained source of truth for oracle bone script corpus records: characters, inscriptions, components, correspondences, provenance, graph edges, and derived research features.

简体中文：
本目录是甲骨文语料记录的人工维护 source of truth，包含甲骨字、卜辞、构件、字形对应、出处、图谱边和派生研究特征。
