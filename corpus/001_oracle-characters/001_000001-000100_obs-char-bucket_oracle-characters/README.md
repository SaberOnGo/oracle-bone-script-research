# Character Bucket 000001-000100 / 甲骨字分桶 000001-000100

English:
This bucket will contain oracle character directories from `obs-char-000001` through `obs-char-000100`.

`001_hust-obc-candidate-packet-manifest.csv` and the generated `01_candidate-character-packet.json` files provide source-marked HUST-OBC candidate metadata for the first 100 reserved IDs. These packets are not accepted oracle-character records and do not make decipherment claims.

简体中文：
本分桶将保存 `obs-char-000001` 到 `obs-char-000100` 的甲骨单字目录。

`001_hust-obc-candidate-packet-manifest.csv` 和生成的 `01_candidate-character-packet.json` 文件为前 100 个保留 ID 提供带来源标记的 HUST-OBC 候选 metadata。这些 packet 不是已接受的甲骨单字记录，也不提出释读结论。
