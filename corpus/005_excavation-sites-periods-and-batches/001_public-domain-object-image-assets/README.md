# Public-domain Object Image Assets / 公共领域藏品图像资产

English:
This directory stores small, source-marked public-domain object images that are safe to commit under the repository `SIZE_LIMIT`. Each image must have a matching metadata file and registry rows in `project_registry/004_asset-source-and-rights-index/`.

Images here are object-level visual evidence and provenance samples. They are not formal paleographic readings, inscription transcriptions, or decipherment conclusions.

简体中文：
本目录存放体积较小、来源标记清楚、符合仓库 `SIZE_LIMIT` 的公共领域藏品图像。每张图片必须有对应 metadata 文件，并在 `project_registry/004_asset-source-and-rights-index/` 中登记来源和权利信息。

这里的图像只是藏品级视觉证据和出处追溯样例，不是正式古文字释读、卜辞释文或破译结论。
