# Project Registry / 项目注册表

English:
`project_registry/` is the first place to understand repository structure, naming rules, project IDs, external source IDs, asset provenance, and bilingual terminology.

简体中文：
`project_registry/` 是理解仓库结构、命名规则、本项目 ID、外部来源 ID、资产出处和中英术语的第一入口。

## Purpose / 用途

- Keep file paths short while preserving source traceability.
- 在保持路径简短的同时保留来源可追溯性。
- Map project-local IDs to external catalogs, databases, plate numbers, old catalog numbers, URLs, and rights status.
- 把本项目 ID 映射到外部字编、数据库、图版号、旧著录号、URL 和权利状态。
- Help humans and AI Agents answer: where did this data come from?
- 帮助人类和 AI Agent 回答：这条资料从哪里来？
