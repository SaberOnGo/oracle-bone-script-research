# Repository Skills / 仓库 Skills

English:
This directory contains repository-local skills for AI Agents. Agents should read the relevant `SKILL.md` before performing data curation, provenance review, or AI evidence-pack work.

简体中文：
本目录保存仓库本地 skills。AI Agent 在执行资料整理、来源复核或 AI evidence pack 工作前，应读取相关 `SKILL.md`。

## Skills / 技能

- `oracle-character-data-curation/`: create or update oracle character records.
- `oracle-character-data-curation/`：创建或更新甲骨单字记录。
- `source-provenance-review/`: verify source references and rights status.
- `source-provenance-review/`：复核来源引用和权利状态。
- `ai-agent-evidence-pack-review/`: prepare and review AI Agent evidence packs.
- `ai-agent-evidence-pack-review/`：准备和复核 AI Agent 证据包。
