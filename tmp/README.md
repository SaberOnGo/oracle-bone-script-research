# Local Temporary Workspace / 本地临时工作区

English:
Use this directory for local-only AI Agent scratch files, downloads in progress, OCR intermediates, cache files, temporary indexes, unpacked archives, and other generated research artifacts that are not ready for Git.

Nothing in this directory should be committed except this `README.md` and `.gitignore`. If a temporary artifact becomes valuable, move only the reviewed, source-marked, size-compliant result into the appropriate project directory.

简体中文：
本目录用于本地 AI Agent 草稿、下载中间文件、OCR 中间产物、缓存、临时索引、解压目录，以及其他尚不适合进入 Git 的研究过程产物。

除本 `README.md` 和 `.gitignore` 外，本目录内容不应提交。如果某个临时产物变得有价值，只能把经过复核、已标注来源、符合尺寸限制的结果移动到对应项目目录。
